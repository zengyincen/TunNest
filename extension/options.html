来源账号与同步方式

微信读书 Gateway API Key（可选）

普通浏览器同步不需要 Key。请先登录微信读书；填写 Key 后会优先使用 Gateway，适合无须打开网页的同步。

豆瓣用户 ID

无需填写 API Key；TunNest 会自动生成豆瓣当前接口要求的请求签名。

豆瓣 Auth Token（仅私密条目需要）
微博用户 UID
微博同步页数

浏览器登录凭据只在对应网站页面内使用，不会上传到 TunNest 或 Cloudflare。GitHub Actions 使用单独配置的 Secrets。
